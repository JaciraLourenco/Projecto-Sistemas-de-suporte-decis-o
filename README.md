# Projecto-Sistemas-de suporte a-decisao
Descrição
Este projeto teve como objetivo o desenvolvimento de um Data Warehouse para a empresa fictícia MediaFlix, no contexto da unidade curricular de Sistemas de Suporte à Decisão. O objetivo foi modelar e preparar uma solução analítica capaz de apoiar a tomada de decisão através da análise de vendas, faturação, lucros e gestão de conteúdos.

O trabalho incluiu a análise do modelo de dados transacional existente, a identificação das fontes de dados relevantes, a criação de melhorias ao modelo relacional, e a implementação de um modelo dimensional em estrela (Star Schema) no Oracle Data Modeler. Foram também gerados os respetivos artefactos técnicos, incluindo diagramas, scripts SQL e relatório do modelo.

O projeto permite analisar informação através de diferentes perspetivas, como país, distrito, cidade, plataforma, género de conteúdo e faixa etária, facilitando assim a criação de relatórios de gestão e o suporte à decisão.
